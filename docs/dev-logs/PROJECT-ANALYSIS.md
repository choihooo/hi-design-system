## 📋 패키지별 상세 분석

### 🎨 Design Tokens (`packages/tokens`)
```bash
4개 파일
├── index.ts          # 토큰 export
├── raw.ts            # 원시값 정의
├── scale.ts          # 스케일 토큰
└── semantic.ts       # 시맨틱 토큰
```
- **특징**: Seed Design KDT 방식 적용
- **색상**: Primary, Secondary, Neutral 계층 구조
- **간격**: 8px 기반의 계층적 간격 시스템
- **타이포그래피**: Display → Body 계층 구조

### 🔧 TypeScript Types (`packages/types`)
```bash
2개 파일
├── index.ts          # 타입 export
└── components.ts     # 컴포넌트 타입
```
- **역할**: 전체 프로젝트의 타입 안정성 보장
- **기능**: 컴포넌트 Props 인터페이스 정의
- **특징**: Strict 모드 활성화

### 🎯 React Components (`packages/react`)
```bash
21개 파일
├── Button/           # 버튼 컴포넌트
├── Input/            # 입력 필드
├── Card/             # 카드 컴포넌트
├── Modal/            # 모달 컴포넌트
└── ...               # 기타 컴포넌트
```
- **특징**: 3가지 사이즈(sm/md/lg) 지원
- **변형**: Primary, Secondary, Ghost, Outline
- **기능**: 접근성, 포커스 상태, 디자인 토큰 적용

### 🎣 Hooks (`packages/hooks`)
```bash
10개 파일
├── useTheme.ts       # 테마 훅
├── useToggle.ts      # 토글 훅
├── useDebounce.ts    # 디바운스 훅
├── useMediaQuery.ts  # 미디어 쿼리 훅
├── useReducedMotion/# 모션 감소 훅
└── useKeyboard.ts    # 키보드 훅 (최신 추가)
```
- **총 10개 커스텀 훅** 개발
- **목적**: 상태 관리 및 부가 기능 제공
- **특징**: TypeScript로 완전히 타입 안전성 보장

### 🎨 Theme System (`packages/theme`)
```bash
5개 파일
├── ThemeProvider.tsx # 테마 제공자
├── ThemeContext.tsx  # 테마 컨텍스트
├── dark-theme.ts     # 다크 테마
├── light-theme.ts    # 라이트 테마
└── index.ts          # export
```
- **기능**: 테마 동적 전환 지원
- **구조**: Context API 기반 구현
- **확장성**: 커스텀 테마 쉽게 추가 가능

### 🎨 Icons (`packages/icons`)
```bash
10개 파일
├── HiButtonIcon.tsx  # 버튼 아이콘
├── HiCardIcon.tsx    # 카드 아이콘
├── HiInputIcon.tsx   # 입력 필드 아이콘
└── ...               # 기타 아이콘
```
- **특징**: 크로스플랫폼 아이콘 시스템
- **기술**: SVG 기반 구현
- **확장성**: 아이콘 쉽게 추가 가능

### 🔧 Presets (`packages/presets`)
```bash
7개 파일
├── index.ts         # 프리셋 export
├── vite/index.ts    # Vite 프리셋
├── vite/plugin.ts  # Vite 플러그인
└── tailwind/       # Tailwind 프리셋 (최신 추가)
```
- **기능**: 개발 환경 설정 자동화
- **특징**: 빌드 최적화 포함
- **확장성**: 프레임워크별 프리셋 추가 가능

## 🔧 기술 스택

### 핵심 기술
- **빌드 도구**: Turbo (병렬 처리, 캐싱)
- **패키지 매니저**: pnpm (workspace 지원)
- **번들러**: Vite (빠른 개발 서버)
- **테스트**: Vitest (단위 테스트)

### 지원 기술
- **타입 시스템**: TypeScript 5.3+
- **CSS-in-JS**: Styled Components 지원
- **접근성**: ARIA 속성, 키보드 내비게이션
- **애니메이션**: Framer Motion 연계 가능성
